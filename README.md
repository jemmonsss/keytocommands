🔑 Core Features

    Custom Keybinds – Players can easily bind any command to a key.
    Remove Keybinds – Delete specific keybinds directly from the UI.
    Dynamic UI – Modern, responsive menu with a sleek design.
    Automatic Keybind Loading – Loads saved keybinds every time a player joins.
    Instant Refreshing – Keybinds update immediately when added or removed.

⚙️ Configuration & Customization

    Fully Configurable – Edit settings in config.lua to match your server’s needs.
    Customizable UI – Modify the HTML, CSS, and JavaScript for a unique look.
    Changeable Button Colors – Adjust UI button colors dynamically.
    Adjustable Menu Title – Personalize the menu’s header text.

📦 Database & Data Handling

    MySQL/OxMySQL Support – Stores keybinds in the database for persistence.
    Save & Load Keybinds – Players’ keybinds remain stored even after restart.
    Fast Keybind Retrieval – Optimized queries ensure instant keybind loading.

🔒 Security & Escrow Protection

    Encrypted Code – Uses FiveM’s escrow system for protection.
    Editable Configuration & UI – Allows customization while securing core logic.
    Server-Side Validation – Ensures keybinds are properly saved and loaded.

📢 Notifications & Logging

    Built-in Chat & Notify System – Sends messages when keybinds are added or removed.
    Logging System – Tracks keybind changes in the console for admin review.

🛠 Additional Features

    Custom Permissions – Restrict keybinding to specific roles if needed.
    Optimized Performance – Minimal impact on server resources.
